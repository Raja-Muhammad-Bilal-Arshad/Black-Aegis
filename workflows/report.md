# Reporting Workflow

## 1. Aggregation
1.  **Collect Artifacts**: Screenshots, tool output logs, hash dumps.
2.  **Timeline**: Create a chronological list of actions.

## 2. Analysis
1.  **Risk Rating**: Calculate CVSS for each finding.
2.  **Business Context**: Why does "SMB Signing Disabled" matter? (Relay attacks).

## 3. Deliverable
1.  **Draft**: Use the `reporting-standards` skill.
2.  **Review**: `validation-reporting-officer` specific check.
3.  **Finalize**: export to PDF/HTML (Encrypted).
